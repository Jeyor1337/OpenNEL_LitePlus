OpenNEL_LitePlus
===========

## 本项目基于基于 [Codexus.OpenSDK](https://github.com/corpnetease/Codexus.OpenSDK) Codexus.Development.SDK 以及 OpenNel_Lite 制作的第三方启动器
### 安装要求
- .Net 9.0

### 安装步骤
1. 载最新的构建结果
2. 双击打开即可

### 怎么装插件？
1. 下载一个codexus 然后在里面安装你要的插件
2. 把codexus的插件复制并放入plugins文件夹
3. 重启程序即可
